# Set an alarm in Clock on iPhone

You can turn your iPhone into an alarm clock, or simply ask Siri to set one for you. Alarms can be set for any time and repeated on specific days of the week. You can also adjust how long the snooze lasts.

**Siri:** [Ask Siri](https://support.apple.com/guide/iphone/aside/summon_siri/26/ios/26) something like, “Set an alarm for 7 a.m.” [Learn how to use Siri](https://support.apple.com/guide/iphone/turn-on-and-activate-siri-iph83aad8922/26/ios/26).

_Note:_ You can also set up a wake up alarm as part of a full [sleep schedule](https://support.apple.com/guide/iphone/set-up-a-sleep-schedule-iphaf56dceb4/26/ios/26) (including bedtimes, wake up times, and more) in the Health app. If you don’t want to set up a sleep schedule, you can set a regular alarm in Clock for the time you want to wake up.
