## Create a custom Focus

If you want to focus on an activity that’s different from any of the provided Focus options, you can create a custom Focus.

![A Focus setup screen for the additional provided Focus options, including Custom, Driving, Fitness, Gaming, Mindfulness, and Reading.](https://help.apple.com/assets/683A54F5227CBBDB98051B96/683A54FC545676DF600F97D4/en_US/86c7dec606ea6379c4de7e58edb3025a.png)

1. Go to Settings ![](https://help.apple.com/assets/683A54F5227CBBDB98051B96/683A54FC545676DF600F97D4/en_US/88b2400e45bcf521514b7252cbb2d959.png) \> Focus.

2. Tap ![the Add button](https://help.apple.com/assets/683A54F5227CBBDB98051B96/683A54FC545676DF600F97D4/en_US/12d48fa9a82b5d7e291da08f1769ba33.png) at the top right, then tap Custom.

3. Enter a name for your Focus, then tap Return.

4. Choose a color and an icon to represent your Focus, then tap Next.

5. Tap Customize Focus, then set up the options for your custom Focus, following the instructions in [Set up a Focus](https://support.apple.com/guide/iphone/set-up-a-focus-iphd6288a67f/18.0/ios/18.0#ipheb6ee12a0).
