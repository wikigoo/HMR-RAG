# Set up a Focus on iPhone

Focus is a feature that helps you concentrate on a task by minimizing distractions. When you want to concentrate on a specific activity, you can customize one of the provided Focus options—for example Work, Personal, or Sleep—or [create a custom Focus](https://support.apple.com/guide/iphone/set-up-a-focus-iphd6288a67f/18.0/ios/18.0#iphf9f22c88c). You can use Focus to temporarily silence all notifications, or allow only specific notifications—ones that apply to your task, for example—and let other people and apps know you’re busy.

**Tip:** To quickly silence all notifications, [open Control Center](https://support.apple.com/guide/iphone/aside/iph7c412b4db/18.0/ios/18.0), tap Focus, then turn on Do Not Disturb.
