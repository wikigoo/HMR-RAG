# How to add an AirTag to Find My

To help keep track of personal items, like keys or a backpack, you can add an AirTag or other compatible Find My network accessories to Find My. You can add items using AirTag setup or the Find My app.

- [Add an AirTag to Find My using AirTag setup](https://support.apple.com/en-us/101602#add-airtag)

- [Add a compatible Find My network accessory or AirTag using the Find My app](https://support.apple.com/en-us/101602#add-compatible)

- [If you're unable to add an AirTag](https://support.apple.com/en-us/101602#unable)

## Add an AirTag to Find My using AirTag setup

To add an AirTag, you need iOS or iPadOS 14.5 or later. To add an AirTag (2nd generation), you need iOS or iPadOS 26.2.1 or later.

1. Hold your AirTag near your iPhone or iPad, then tap Connect. If you have more than one AirTag and get the alert "More than one AirTag detected," make sure that you bring only one AirTag near your device at a time.
![iPhone screen showing Connect AirTag button.](https://cdsassets.apple.com/live/7WUAS350/images/icloud/find-my/ios-26-iphone-16-pro-home-screen-airtag-connect.png)
2. Select a category to name your AirTag. If you choose Custom Name, select an emoji.
![iPhone screen showing how to select a name for AirTag.](https://cdsassets.apple.com/live/7WUAS350/images/icloud/find-my/ios-26-iphone-16-pro-home-screen-airtag-connect-name-airtag.png)
3. To link your AirTag with your Apple Account, tap Agree or Continue again.

4. Tap Done.
