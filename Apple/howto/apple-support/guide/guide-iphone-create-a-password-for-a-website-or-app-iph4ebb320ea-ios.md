# Create a password for a website or app on iPhone

Your iPhone can generate a password to use on a website or app.

1. Go to the Passwords app ![](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/10c7d2493aff94aeeba904e260e6c1ec.png) on your iPhone.

2. Tap All, then tap ![the Toolbar New Password button](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/0d07cb7185e2d87af8a9ad8b4ace6995.png) in the lower-right corner.

3. Enter details about the website or app, and a user name for your account.

4. Tap password.

A strong password is suggested for you.

5. Tap the suggested password (or enter one).

6. Tap ![the Save button](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/8cf38aa668e293079ba389088a78ec31.png).


You can also create a password when you sign up for services on websites and in apps. See [Create a strong password for a new account](https://support.apple.com/guide/iphone/automatically-fill-in-strong-passwords-iphf9219d8c9/26/ios/26#iph04c1f176b).

**See also** [Change the password for a website or app on iPhone](https://support.apple.com/guide/iphone/change-the-password-for-a-website-or-app-iph9e39791bd/26/ios/26)
