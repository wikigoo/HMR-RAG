# Get started with Reminders on iPhone

The Reminders app ![](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/ec6119fc5bf93b83f78120434ec8118d.png) helps you remember anything you need to track. You can set reminders, make grocery lists, create checklists for projects, and organize tasks in to-do lists.

![The Reminders app, showing a new task being added to the Reminders list.](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/0eeb776c501dbf8e6ae3dca59448aee0.png)

## Set a reminder

Simply type, dictate, or ask Siri when you need to remember something.

![The Reminders app, showing a grocery list with items in categories such as Produce and Seafood.](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/6c7cafc2cd5938942ec4dcfa22bfa24f.png)

## Create a grocery list

Make your shopping easier with a grocery list that automatically sorts items into categories. To convert a regular list into a grocery list, tap ![the More button](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/aad05ec99d91fb0497859fc9c0c181fb.png), tap Show List Info, tap List Type, then choose Groceries.

![The Reminders app, showing a grocery list with options for sharing at the bottom.](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/ee52fbbc2a9473564bcc9c6cbd5ae767.png)

## Collaborate on a list

To invite people to work on a list, tap ![the Share button](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/748a151e2276ad5df4d03ec8d4506bf7.png), then choose a sharing option.

![The Reminders app, showing multiple lists and the Tag Browser at the bottom.](https://help.apple.com/assets/69F8EBBDF3B89A4F6E0C704C/69F8EBC43862495245036393/en_US/a679f4f2abfa1a97772ee7008ed99ebc.png)

## Organize your reminders

You can create new lists to suit your needs, categorize with tags, and manage your reminders with Smart Lists.

_Note:_ All Reminders features described in this guide are available when you use [updated reminders in iCloud](https://support.apple.com/guide/iphone/add-or-remove-accounts-iph8739025dd/26/ios/26#iph8bbc4e260). Some features aren’t available when using other accounts.
