# What if phone storage space is not enough?
1. Uninstall the infrequently used App.  2. For mobile phones that support SD card, it is recommended to put media files (videos, music, etc.) on the external memory card.  3. Regularly clean the downloaded files (software installation package).  4. Regularly clean up the temporary files of the system (cache).  5. Enter the security center for garbage removal.
